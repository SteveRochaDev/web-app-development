# GatewayShop&Trade

Full-stack web application developed as part of the **Web App Development** course at the University of Algarve (UAlg).

This project is a fictional online marketplace built as a group assignment, focusing on frontend and backend development.

## About the Project

A fictional online marketplace developed by Gonçalo C., Steve R., and Tiago L.

The project explores the design and implementation of a full-stack web application, including user interface development, backend logic, and data handling.
